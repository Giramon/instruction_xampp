Пошаговая инструкция по использованию

Без базы данных С базой данных

Инструкция если нет базы данных

Шаг 1. Архивируем всю папку с проектом

Шаг 2. Выбираем ZIP архив (обязательно)

Шаг 3. Выбираем zip архив

Шаг 4. Вводим название zip архиву

Инструкция если база данных MySQL

Шаг 1. Откройте MySQL -> Admin

Шаг 2. Выберите вашу базу данных в списке

Шаг 3. Выберите пункт операции

Шаг 4. Напишите ваше название, после этого "_" и ваш ID (fmv_17234)

Шаг 5. Измените в коде название из прошлого пункта с ID

Шаг 6. В верхнем меню выберите "Экспорт"

Шаг 7. Формат оставляете SQL и нажимаем "Экспорт"

Шаг 8. Помещаем файл в папку с проектом

Шаг 9. Архивируем всю папку с проектом

Шаг 10. Выбираем ZIP архив (обязательно)

Шаг 11. Выбираем zip архив

Шаг 12. Вводим название zip архиву

Шаг 13. Вводим название файла и базы данных

Шаг 14. Вводим ваш ID
